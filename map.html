Beginners

Follow the 4 step process below to start mapping, this takes around 30-45 minutes. Remember you can pause and rewind the videos as you go along.

Step 1 - Create an Open Street Map Account

Go to the OpenStreetMap website and register to create an account, follow the link in the email they send to confirm your account.

Return here once complete.

Step 2 - Find a task to Map

Go to the Task Manager to find a project and task that interests you.

Learn how to lock a task to edit, select an editing program and change the background settings

iD editor is recommended for beginners.

Step 3 - Learn to map buildings and roads

Once you've found a task, learn to map building and roads.

Training for other landmarks is available on the OSM website .

Step 4 - Save and complete your task

It's important to save regularly, if you have complete the task mark it as complete, if it's not complete unlock it for others to complete, remember to leave a message for the next mapper.
